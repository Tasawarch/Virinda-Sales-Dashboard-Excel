Project Name: Virinda sales Data Analysis & Visualization in Excel
**Description:**
This project focuses on cleaning and analyzing raw sales data using Microsoft Excel. The goal was to uncover key business insights through data preprocessing, pivot tables, and interactive dashboards.
**Key Steps & Tasks:**
Performed data cleaning including removing duplicates, handling missing values, and formatting inconsistencies.
Created Pivot Tables to generate insights such as:
Orders vs Sales performance
Sales by Gender (Male vs Female)
Orders by Age & Gender
Order Status Analysis (Delivered, Canceled, Refunded, Returned)
Top 10 States by Orders & Sales Channels
Added interactive Slicers for filtering by Month, Channel, and Category.
Built a dynamic Excel dashboard to visualize trends and make business decisions easier.
**Tools Used:**
Microsoft Excel (Pivot Tables, Slicers, Charts)
Data Cleaning & Formatting Techniques
**Outcome:**
Identified sales trends and customer behavior patterns that can help improve marketing and operational strategies.
